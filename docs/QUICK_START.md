## 项目简介

### 什么是 DataSec Hub？

DataSec Hub 是一个专注于**数据安全合规**的知识服务平台，帮助企业：

✅ 查询和检索数据安全相关的法律法规  
✅ 通过结构化的COU（合规义务单元）理解合规要求  
✅ 构建适合自己业务场景的合规方案  
✅ 通过API集成到现有系统  

### 核心概念

**COU (Compliance Obligation Unit) - 合规义务单元**
- 将复杂的法律条款转化为清晰的合规义务
- 每个COU都有明确的：
  - 义务类型（禁止性/强制性/推荐性/指导性）
  - 要求的行动
  - 适用范围（行业/区域/企业类型）
  - 权重（优先级）

**三层权重系统**
- **基础权重**：基于政策层级（法律10分 → 指南4分）
- **场景权重**：基于标签匹配度
- **最终权重**：综合权重，帮助确定优先级

## 核心功能

### 功能一：查找合规义务

**场景**：想知道"数据出境"相关的合规要求

**操作步骤**：
1. 进入"COU浏览器"
2. 在右侧筛选面板：
   - 搜索框输入"数据出境"
   - 或点击"跨境传输"标签
3. 查看匹配的COU列表
4. 点击任意COU查看详情

**结果**：
- 看到所有与数据出境相关的合规义务
- 了解每个义务的权重和优先级
- 查看具体的行动要求

---

### 功能二：构建合规场景

**场景**：游戏公司要出海到欧盟

**操作步骤**：
1. 进入"场景构建器"
2. 选择"从模板创建" → "游戏出海场景"
3. 配置核心要素：
   - 行业：游戏
   - 目标区域：欧盟
   - 企业类型：中小企业
4. 添加特殊要求（可选）
5. 点击"分析场景"

**结果**：
- 看到权重分布图表
- 获得匹配的COU列表（按优先级排序）
- 了解需要重点关注的合规要求

---

### 功能三：版本管理

**场景**：某个法律有新版本发布

**操作步骤**：
1. 进入"政策浏览器"
2. 找到对应的政策
3. 点击"版本历史"
4. 查看不同版本的变化

**结果**：
- 了解政策的演变历史
- 查看新旧版本的对比
- 确保合规方案使用最新版本

---

### 功能四：API集成

**场景**：将合规知识集成到企业内部系统

**操作步骤**：
1. 进入"数据看板"
2. 点击"API密钥"标签
3. 创建新的API密钥
4. 进入"API文档"查看接口说明
5. 复制代码示例
6. 在自己的系统中调用API

**API示例**：
```javascript
// 获取COU列表
const response = await fetch('https://api.datasec-hub.com/api/v1/cous', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});
const cous = await response.json();
```
